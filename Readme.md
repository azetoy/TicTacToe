Nouh Master et Zineddine Berrichi

Notre projet était de créer le jeu tic-tac-toe (Morpion) en Java en utilisant ce que l'on a appris en cours (classe, héritage etc.).

Pour lancer il suffit d'ouvrir le dossier avec éclipse et de cliquer sur exécuter/compiler, ou bien avec le jar double-cliqué sur le .jar

Ensuite un menu s'affiche et vous demandez le choix entré ->
Pve : dont le but est de battre lune pseudo IA il vous suffira de cliquer avec votre souris pour placer votre symbole 
Pvp : dont le but est de battre un joueur qui joue contre vous sur le même ordi au tour part tour

                                                                                                          
Le but du jeu est simple dans un quadrillage de 3*3 il vous faut aligner sur la même ligne ou colonne ou diagonale votre symbole.

ATTENTION le .jar ne fonctionne que dans l'archive/le dossier morpion merci de ne pas le déplacer mais vous pouvez déplacer l'archive en entière étant donné qu'il prend le chemin absolu jusqu'à sa position actuelle
